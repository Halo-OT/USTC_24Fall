# 实验报告：回文字符串判断程序

## **Purpose**

​	本实验旨在使用 LC-3 汇编语言实现一个回文字符串判断程序。回文字符串是指从前向后和从后向前读取都相同的字符串，例如 `"level"` 或 `"madam"`。本程序接收一个字符串和其长度，通过比较字符串首尾字符，判断其是否为回文，并将结果存储到指定的内存地址。

------

## **Principles**

1. **回文字符串判断的基本原理**：

   - 从字符串的首尾开始，逐字符比较，若首尾字符均相等，则继续比较下一个。
   - 若出现任意一对字符不相等，则字符串不是回文。
   - 当首尾指针交叉（即首指针超过尾指针）时，字符串被确认是回文。

2. **程序逻辑**：

   - 使用寄存器 `R1` 和 `R2` 分别作为首指针和尾指针。

   - `R4` 和 `R5` 用于加载当前字符，比较其是否相等。

   - 一个标志寄存器 

     ```
     R3
     ```

      用于记录判断结果：

     - 若字符串是回文，则 `R3 = 1`。
     - 若不是回文，则 `R3 = 0`。

   - 结果存储在指定地址 `x3200`。

3. **边界条件**：

   - 空字符串（长度为 0）默认是回文。
   - 单字符字符串天然是回文。

------

## **Procedure**

### **实验步骤**

1. **初始化内存和寄存器**：

   - 从地址 `x3100` 读取字符串长度到寄存器 `R0`。
   - 若长度为 0，直接判断为回文，结果设为 `1` 并存储。

   ```
   				LDI R0, LENGTH      ; R0 = N (字符串长度)
           BRz DONE1
           LD R1, STRING      ; R1 = 起始指针
           ADD R2, R1, R0     ; R2 = R1 + N (字符串末尾之后的位置)
           ADD R2, R2, #-1    ; R2 = 字符串末尾指针
   
           AND R3, R3, #0     ; 清零 R3 (用于存储比较结果)
   ```

   

2. **循环比较字符串**：

   - 设置两个指针：`R1` 指向字符串起始地址，`R2` 指向字符串末尾地址。
   - 使用 `LDR` 指令加载首尾字符到 `R4` 和 `R5`，然后通过减法比较是否相等。
   - 若首尾字符相等，移动指针继续比较；否则，设置结果为 `0` 并跳转结束。

   ```
   LOOP    LDR R4, R1, #0     ; R4 = M[R1] (当前字符)
           LDR R5, R2, #0     ; R5 = M[R2] (对应末尾字符)
           
           NOT R5, R5         ; R5 = ~M[R2]
           ADD R5, R5, #1     ; R5 = -M[R2]
           ADD R4, R4, R5     ; 比较字符是否相等，R4 = M[R1] - M[R2]
   
           BRz CONTINUE       ; 如果相等，继续
           AND R3, R3, #0     ; 如果不相等，将 R3 置为 0，表示不是回文
           BRnzp DONE2         ; 跳转到结束
   ```

   

3. **终止条件**：

   - 当首指针超过尾指针（`R1 > R2`）时，比较结束，字符串为回文。

   ```
   CONTINUE ADD R1, R1, #1     ; R1++
           ADD R2, R2, #-1    ; R2--
           
           NOT R5, R2
           ADD R5, R5, #1      ; R5 = -R2
           ADD R6, R1, R5      ; R6 = R1 - R2 <= 0
           
           BRn LOOP           ; 如果 R1 <= R2，继续循环
   ```

   

4. **存储结果**：

   - 将 `R3` 中的判断结果存储到 `x3200`。

   ```
   DONE1   LD R7, RESULT
           ADD R3, R3, #1     ; 假设字符串是回文，R3 = 1 (FLAG)
           STR R3, R7, #0
           
   
   DONE2   LD R7, RESULT      ; 加载结果地址
           STR R3, R7, #0     ; 将 R3 的值存入结果地址 (1: 是回文，0: 不是回文)
   
           TRAP x25           ; 结束程序
   ```

   

#### 所有代码：

```
.ORIG x3000

        STR R3, R7, #0

```

### **遇到的错误与解决方法**

1. **空字符串的处理**：
   - **问题**：未考虑长度为 0 的字符串，程序会进入循环执行无意义的比较。
   - **解决**：在读取长度后，增加检查条件 `BRz DONE1`，直接将结果设为 `1` 并跳转结束。
2. **没别的问题了（）**
